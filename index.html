What is D-Verse?

D-Verse is a decentralized framework for building capsule-powered applications that execute securely and independently on user devices. Designed for trust-aware logic, offline-first operations, and verifiable data, it reimagines how digital systems work without central servers or cloud lock-in.

Core Modules

kapsule-rt

Local WASM runtime for executing signed logic securely on-device.

dverse-dag

Trust-aware DAG for tracking capsule lineage and validating identities.

dverse-did

Decentralized identity layer with keypair generation and signature verification.

dverse-net

P2P sync with DHT-based routing and optional relay support.

dverse-relay

Federated relay nodes for NAT traversal, moderation, and sync caching.

dverse-storage

Encrypted, hash-addressed blob storage for portable content sharing.
