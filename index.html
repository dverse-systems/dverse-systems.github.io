What is d-verse?

d-verse is a sleek, decentralized framework for crafting secure, capsule-powered apps that run directly on user devices. Designed for trust-aware logic, offline-first operations, and verifiable data, it redefines digital systems without central servers or cloud infrastructure.

Core Modules

dverse-runtime

Secure WASM runtime for executing signed logic securely on-device.

dverse-dag

Trust-aware DAG for tracking capsule lineage and validating identities.

dverse-did

Decentralized identity layer with keypair generation and signature verification.

dverse-net

P2P sync with DHT-based routing and optional relay support.

dverse-relay

Federated relay nodes for NAT traversal, moderation, and sync caching.

dverse-storage

Encrypted, hash-addressed blob storage for portable content sharing.
